# COVID-19 Global Analytics Dashboard

## Overview

This project analyzes global COVID-19 data using Python, SQL, and Power BI.

The dashboard provides insights into:

- Total Cases
- Total Deaths
- Mortality Rate
- Country Comparisons
- Global Trends

## Tools Used

- Python
- Pandas
- SQL
- Power BI

## Dataset

Source: Our World In Data (OWID)

## Dashboard Preview

![Dashboard](Images/dashboard.png)

## Key Features

- Interactive country filter
- KPI cards
- Trend analysis
- Top 10 countries by cases
- Top 10 countries by deaths
- Mortality rate analysis
